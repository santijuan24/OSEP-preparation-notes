# OSEP Preparation Notes

📚 Notas de preparación para exámenes de ofensiva en seguridad informática y máquinas de práctica.

## 📖 Contenido

Este repositorio contiene apuntes detallados sobre:

- **Máquinas de Laboratorio**: Writeups y notas de resolución de máquinas
- **Técnicas de Hacking**: Procedimientos, comandos y metodologías
- **Análisis de Vulnerabilidades**: Reconocimiento, explotación y post-explotación
- **Herramientas**: Configuración y uso de herramientas de pentesting
